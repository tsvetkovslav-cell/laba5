Десктопное приложение на C++/CLI (Windows Forms) для учёта клиентов и типов вкладов банка. Использует SQLite для хранения данных.

## Возможности

- **Клиенты** — добавление, изменение, удаление
- **Типы вкладов** — название, процентная ставка (0–1 000 000%), категория
- **Вклады** — открытие вклада клиенту с выбором типа и суммы
- **Расчёты** — процент по клиенту, общая сумма процентов, топ-клиент по процентам
- **База данных** — SQLite (bank.db), экспорт/импорт, очистка
- **Файлы** — сохранение и загрузка в текстовый файл (.txt)
- **Сортировка** — клиенты по имени

### Категории вкладов

1. Краткосрочный  
2. Долгосрочный  
3. Специальный  
4. Накопительный  

## Требования

- Windows 10/11
- Visual Studio 2022 (v143) с поддержкой C++/CLI
- .NET Framework 4.7.2
- System.Data.SQLite (NuGet)

## Сборка

1. Откройте `BD66.sln` или `BD66.vcxproj` в Visual Studio.
2. Восстановите пакеты NuGet:  
   ПКМ по решению → Restore NuGet Packages.
3. Выберите конфигурацию (Debug/Release) и платформу (x64 или Win32).
4. Соберите проект (Build → Build Solution или F7).

Исполняемый файл появится в `x64\Debug\BD66.exe` или `x64\Release\BD66.exe`.

## Архитектура

- **Bank** — Singleton, управляет клиентами и типами вкладов
- **Client** — данные клиента и привязанный тип вклада
- **DepositType** — тип вклада (название, ставка %, категория)

Подробнее см. `ClassDiagram.md` или `ClassDiagram.puml`.

## Использование

При первом запуске, если `bank.db` нет, создаётся демо-данные. При успешной загрузке базы используются сохранённые данные.

- **Клиенты** — работа вкладки «Клиенты»
- **Типы вкладов** — вкладка «Типы вкладов»
- **Открыть вклад** — клиент + тип вклада + сумма
- **Информация БД** — вкладка «Информация БД» (экспорт, импорт, очистка)
